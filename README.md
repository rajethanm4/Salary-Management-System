# Salary Management System

A full-stack web application for managing employee salaries, payroll, and records.

## 🚀 Tech Stack

* Backend: Ruby on Rails (API)
* Frontend: React (Vite)
* Database: PostgreSQL
* Authentication: JWT

## ✨ Features

* Employee management
* Salary structure & payroll processing
* Secure authentication (login/signup)
* RESTful API architecture
* Scalable frontend with React

## 🧩 Architecture

* Rails serves as a JSON API
* React handles UI and client-side logic
* Communication via REST APIs

## 📌 Future Enhancements

* Payslip generation (PDF)
* Role-based access control (Admin/Employee)
* Dashboard with analytics
* Email notifications
